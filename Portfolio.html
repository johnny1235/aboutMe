Nodes Application

Created using PHP, HTML5/CSS , Javascript, JQuery, pgSQL and Apache run on a linux server.

This application requires users to be authenticated securely with LDAP. The purpose of this web application is to allow users to efficiently route toll free numbers and to allow them to freely design how these calls are handled. Based on inputs in the GUI interface, the web application will generate a XML and a formatted VXML file used by our systems to determine how to handle the call (prompting, transfer or etc.).
